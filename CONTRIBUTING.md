# Contributing

Thank you for helping keep this list accurate and useful!

## How to add a new source

1. Open an issue or submit a pull request.
2. Provide the following information:
   - **Name** of the source
   - **URL** (if still available)
   - **Category** (e.g., Movies, Music, Software, Academic)
   - **Why it's legal** (public domain, Creative Commons, copyright-holder, etc.)
   - **How you verified it** (e.g., tested the URL, checked licensing info on the site)

## How to update a source status

If you find that a listed source is no longer available, or has changed URL, please:

1. Check that the source is truly offline (try multiple times over a few days).
2. Submit a pull request updating its status and adding a note.

## Criteria for inclusion

A source should be included only if:

- The content distributed via BitTorrent is **unambiguously legal** (public domain, Creative Commons, open-source, or explicitly authorized by the copyright holder).
- The website or tracker is **primarily or substantially** focused on legal content.

Sources that mix legal and potentially copyrighted content (like general-purpose torrent sites) are **not** included, even if some of their content is legal.

## Format

Sources are listed in tables with the following format:

```
### Source Name

|              |                                              |
| ------------ | -------------------------------------------- |
| **Website**  | [https://example.com](https://example.com)   |
| **Status**   | ✅ Active / ❌ Offline                        |
| **Category** | Type of content                              |

Description of the source...
```

Add new sources in the appropriate category section, keeping alphabetical order.

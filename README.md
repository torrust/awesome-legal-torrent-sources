# Awesome Legal Torrent Sources

A curated, community-maintained list of websites that distribute **legal, public-domain, and openly-licensed content via BitTorrent**.

BitTorrent is often associated with piracy, but the protocol itself is simply a peer-to-peer file-sharing technology. Many creators, archives, and open-source projects use it to distribute content that is genuinely free to share — public-domain movies, Creative Commons music, Linux distributions, academic datasets, and more.

This list aims to be a **bootstrap entry point**: a quick, verified starting point for anyone looking to find legal torrents. Sources are checked periodically and marked with their availability status.

## The Challenge

The main source of friction for legal torrenting is **discovery**. Unlike centralized streaming platforms, BitTorrent has no built-in catalog. This list bridges that gap — not as a central authority, but as a community-maintained directory you can trust, fork, and contribute to.

## Cultural Archives

### Internet Archive

|              |                                                                                                |
| ------------ | ---------------------------------------------------------------------------------------------- |
| **Website**  | [https://archive.org](https://archive.org)                                                     |
| **Torrents** | [https://archive.org/details/bittorrent](https://archive.org/details/bittorrent)               |
| **Status**   | ✅ Active                                                                                      |
| **Category** | Books, movies, music, software, datasets                                                       |

One of the largest digital libraries in the world. Offers millions of free books, movies, music recordings, software packages, and datasets, many available via BitTorrent. The most comprehensive source of unquestionably legal torrents on the internet.

|              |                                                             |

Provides more than 70,000 free public-domain ebooks. Some collections are distributed via torrents. The oldest digital library of freely redistributable literary works.

## Finding More Sources

Beyond the curated list, you can discover additional sources by searching for:

- `public domain torrent`
- `creative commons torrent`
- `open source software torrent`
- `linux iso torrent`
- `academic torrents`
- `internet archive torrent`

Always verify that content is genuinely in the public domain or openly licensed before downloading or redistributing.

## Why a Repo Instead of a Blog Post?

A curated list like this needs to be **community-maintained** to stay relevant. Sources go offline, new ones appear, and availability changes. A GitHub repo allows anyone to submit updates via pull requests, keeping the information current far better than a static blog post ever could.

The [Torrust website](https://torrust.com) includes this list in expanded form as a [blog post](https://torrust.com/blog/sources-of-public-domain-torrents), cross-referencing this repo as the canonical, up-to-date source.
